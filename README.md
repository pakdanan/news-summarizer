# AI News Summarizer

Aplikasi web untuk meringkas teks berita menggunakan Generative AI.

## Fitur
- Input teks berita
- Ringkasan otomatis
- Responsif untuk semua device
- Copy hasil ringkasan

## Cara Menggunakan
1. Tempel teks berita ke textarea
2. Klik tombol "Ringkas Sekarang"
3. Salin hasil ringkasan

## Teknologi
- Alpine.js untuk interaktivitas
- Tailwind CSS untuk styling
- Hosting di GitHub Pages
